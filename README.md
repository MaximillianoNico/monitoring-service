# monitoring-service
[LittleLive]: Monitoring Service API 

## How to run
> First need to install all dependencies by using 
```
yarn install
```
> Execute Migration
```bash
npx knex migrate:latest --knexfile knexfile.ts
```
> Then, start the service 
```
yarn dev
```
